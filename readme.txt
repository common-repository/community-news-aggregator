A plugin for importing external content from members' blogs to a community blog.

== Description ==

A community blog has to be updated with appropriate content to keep their readers in sync about what happens inside a community. But, when members of a community are busy with their other/personal projects, there is no time to post something on the community's blog exclusively.
Community News Aggregator plugin gives a community blog, power to synchronize their content with their members' blog content. It lets the blog admins to update their blogs for latest feeds from the blogs configured by their registered members. 

NOTE: A member who has configured its blog with this plugin can mark the posts for sync by posting it to a category name same as the community label.

= Features =

* Members/Readers registered as Subscribers can configure their WordPress or Blogger blogs for giving in updates.
* Manual Update to be used by Blog Admins in the Admin Panel.
* Community Settings(for setting Community Label)
* The aggregated blog posts are created in the community blog's post database with a 'pending' status. The admin can either publish it or delete it.

== Notes ==

For the admins, its important to set the Community Label in the Community News Aggregator->Settings page in the admin panel for aggregation. Set it to nothing for no aggregation.

== Installation ==

* Place your plugin directory into wp-content/plugins folder.
* Activate the plugin from the admin panel.
* Set a Community Label to let the users bring content to the community's blog.
